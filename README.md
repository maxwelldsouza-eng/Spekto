# Spekto
Real-time property inspection marketplace connecting Clients with Scouts for on-demand video captures.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Database: Supabase (PostgreSQL)
- Authentication: Supabase Auth
- Storage: Supabase Storage
- Payments: Stripe Connect
- Code: Google AI Studio

## Folder Structure
- /config — configuration files
- /auth — login, register, verify email, role selection
- /client — client dashboard, new inspection, billing
- /scout — scout dashboard, earnings
- /shared — inspection detail view
- /database — schema and database functions
- /assets — CSS, images
